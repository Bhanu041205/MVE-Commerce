User Registration and Login flow with JWT token generation

sequenceDiagram actor U as User Browser participant R as React Frontend participant A as AuthController participant S as AuthService participant DB as MySQL Database participant J as JWT Provider rect rgb(232, 245, 233) Note over U,J: USER REGISTRATION FLOW U->>R: Fill registration form R->>A: POST /auth/register A->>S: registerUser(dto) S->>DB: Check if email exists DB-->>S: Email not found - OK S->>S: Encrypt password with BCrypt S->>DB: Save new User with role CUSTOMER DB-->>S: User saved successfully S-->>A: Return success message A-->>R: 200 OK Registration successful R-->>U: Redirect to Login page end rect rgb(227, 242, 253) Note over U,J: USER LOGIN FLOW WITH JWT U->>R: Enter email and password R->>A: POST /auth/login A->>S: loginUser(dto) S->>DB: Find user by email DB-->>S: User record found S->>S: Verify password with BCrypt S->>J: Generate JWT token with HS256 J-->>S: Signed JWT token returned S-->>A: Return token + UserDTO A-->>R: 200 OK with token and user info R->>R: Store token in Redux and localStorage R-->>U: Redirect to Home or Admin Dashboard end rect rgb(255, 243, 224) Note over U,J: AUTHENTICATED API REQUEST U->>R: Navigate to My Orders R->>A: GET /orders with Authorization Bearer JWT A->>J: Validate token signature J-->>A: Token is valid A->>DB: Load user details for RBAC DB-->>A: User details loaded A-->>R: 200 OK with orders list R-->>U: Display orders on screen end
